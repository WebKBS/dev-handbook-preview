steps() 함수 예시 (이동 / 타이핑 / 스프라이트)

steps(n)은 진행을 n단계로 쪼개서 “뚝뚝 끊어” 보이게 합니다. 프레임 애니메이션, 타이핑 효과, 스프라이트 시트에 자주 씁니다.

1) 계단식 이동

steps(5) vs linear

같은 100px 이동을 1초 동안 수행하지만, steps는 “점프”처럼 보입니다.

2) 타이핑 효과

steps(40)로 “타닥타닥”

너비 증가를 steps로 끊어서 글자가 한 글자씩 나오는 느낌을 냅니다.

console.log("typing with steps(40)...");
3) 스프라이트 애니메이션

steps(10)으로 프레임 넘기기
